Folder Descriptions:

- gis_feature_engineering: Scripts for creating release risk and well exposure datasets. Additional materials as well.

- imputation: Scripts for imputing lab report extraction, and creating base data for models. Should be run before gis_feature_engineering or any models/analyis.

- Lab report extraction: Code for extracting lab reports

- Release risk model: Code for release risk modeling.

- source attribution analysis: Code for source attribution analysis.

- Tableau Workbooks: Additional visualizations of models

- Well exposure model: Code for well exposure model creation.

Installation instructions:
1) Open command/anaconda prompt 
2) cd 'C://path_to_PFAS-Analysis_folder'
3) pip install -r requirements.txt
